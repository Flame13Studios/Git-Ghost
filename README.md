
# 👻 Git-Ghost

A local-first background development ledger and automated commit summary generator built in Python.

## 🚀 How to Run It
1. Double-click `start_ghost.bat` to open your live dashboard window.
2. Code normally. It logs your changes automatically!

## 📊 Dashboard Commands
* `python ghost.py` — View your work session history.
* `python ghost.py --commit` — Create an automated Git commit summary for today.
* `python ghost.py --export notes.md` — Save your history as a document.
